# oc-p05-sql
Base de données relationnelle SQL pour analyse du marché immobilier français (S1 2020). Schéma 3NF avec 12 requêtes analytiques complexes : segmentations régionales, TOP départements, évolutions trimestrielles, prix/m² par typologie. Démontre maîtrise SQL (jointures, agrégations, CTEs), modélisation de données et nettoyage CSV/Power Query
